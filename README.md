We ran into an issue with npm install, and ran the following commands to get past it:

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.38.0/install.sh | bash

command -v nvm

nvm install --lts

nvm ls

npm --version

npm install
